# Contributing Guide

Thanks for your interest in improving the Atelier documentation.

## Building locally

This is a documentation site. Before opening a pull request:

1. Install the documentation toolchain (see the build configuration in this repo).
2. Serve the docs locally and verify your pages render.
3. Check that links resolve and code samples are runnable.

## Style

- Keep pages focused and skimmable.
- Prefer runnable, copy-pasteable examples.
- Match the existing structure and tone.

## Reporting an issue

Open an issue describing the page, the problem, and the suggested fix.
