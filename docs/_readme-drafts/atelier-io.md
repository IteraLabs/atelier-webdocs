# atelier-io

Persistence for the Atelier SDK: Parquet, CSV, and JSON readers and
writers for every top-level type — orderbooks, trades, funding rates,
liquidations, open interest, and aggregate snapshots. Optional
extension traits (`FlushToParquet`, etc.) plug into worker pipelines
when the `connect` and `parquet` features are enabled.
